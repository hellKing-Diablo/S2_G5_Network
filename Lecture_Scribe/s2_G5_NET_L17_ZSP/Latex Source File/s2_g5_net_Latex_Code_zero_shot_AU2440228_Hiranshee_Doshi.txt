\section{Motivation: Why Study Joint Random Variables}

In many practical systems, multiple random variables must be analyzed simultaneously rather than individually. Studying only one random variable may fail to capture the relationship between variables.

\subsection{Smart Transportation Example}

Two random variables are considered:

\[
X = \text{Vehicle Speed}
\]

\[
Y = \text{Traffic Density}
\]

Key question:

Can the speed of vehicles be high when traffic density is also high?

Applications include:

\begin{itemize}
\item Adaptive traffic signal control
\item Congestion prediction
\item Autonomous vehicle routing
\end{itemize}

\subsection{Wireless Network Performance Example}

Two variables in wireless communication systems:

\[
X = \text{Signal-to-Noise Ratio (SNR)}
\]

\[
Y = \text{Data Throughput}
\]

Observation: Throughput depends on SNR, but the relationship is not deterministic.

Question:

If SNR is high, will throughput always be high?

Applications:

\begin{itemize}
\item 5G scheduling
\item Adaptive modulation
\item Network planning
\end{itemize}

\subsection{Weather Prediction Example}

Random variables:

\[
X = \text{Rainfall}
\]

\[
Y = \text{Temperature}
\]

Weather forecasting models must analyze these variables jointly.

\subsection{Drone Delivery Example}

Random variables:

\[
X = \text{Wind Speed}
\]

\[
Y = \text{Delivery Time}
\]

Application: Logistics and UAV delivery systems.

\subsection{Dice Example}

Consider rolling two dice.

\[
X = \text{Value on first die}
\]

\[
Y = \text{Value on second die}
\]

Example questions:

\[
X + Y = 7
\]

\[
X > Y
\]

These events require joint probability calculations.

\section{Discrete Case -- Joint PMF}

For discrete random variables $X$ and $Y$, the joint probability mass function is written as

\[
P(X=x, Y=y)
\]

\subsection{Joint PMF Table Representation}

Joint probabilities can be represented in a table.

\begin{center}
\begin{tabular}{c|c}
Row & Values of $X$ \\
Column & Values of $Y$ \\
\end{tabular}
\end{center}

Each table entry represents:

\[
P(X=x_i, Y=y_j)
\]

This table summarizes the joint probability distribution.

\section{Example: Rolling Two Dice}

Let

\[
X = \text{Outcome of die 1}
\]

\[
Y = \text{Outcome of die 2}
\]

Each ordered pair $(X,Y)$ represents a possible joint outcome.

Example events:

\[
X + Y = 7
\]

\[
X > Y
\]

These probabilities are computed by summing the probabilities of all pairs satisfying the condition.

\section{Continuous Case -- Joint PDF}

For continuous random variables, we define the joint probability density function (PDF):

\[
f_{X,Y}(x,y)
\]

\subsection{Geometric Interpretation}

For continuous variables:

\begin{itemize}
\item The joint distribution is visualized over a 2D region.
\item Probability corresponds to the area under the surface.
\end{itemize}

Probability over a region $R$ is:

\[
P((X,Y) \in R) = \iint_R f_{X,Y}(x,y)\,dx\,dy
\]

\section{Joint Cumulative Distribution Function (Joint CDF)}

\subsection{Definition}

The joint cumulative distribution function is defined as:

\[
F_{X,Y}(x,y) = P(X \leq x, Y \leq y)
\]

This represents the probability that both events occur simultaneously.

\section{Continuous Case of Joint CDF}

For continuous variables:

\[
F_{X,Y}(x,y) =
\int_{-\infty}^{x}\int_{-\infty}^{y} f_{X,Y}(u,v)\,dv\,du
\]

\section{Discrete Case of Joint CDF}

For discrete variables:

\[
F_{X,Y}(x,y) =
\sum_{x_i \le x} \sum_{y_j \le y} P(X=x_i, Y=y_j)
\]

This represents the total probability of all pairs satisfying

\[
X \le x, \quad Y \le y
\]
