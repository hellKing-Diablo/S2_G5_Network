This file explains the idea of spatial throughput in wireless networks using simple probability concepts. It shows how random device placement, signal fading, and interference affect network speed and coverage.

The file compares Aloha (random access) and DPP (smart scheduling) to see which method gives better throughput by choosing the right number of active devices. It mainly focuses on understanding the problem, assumptions, uncertainty, and basic simulation flow.

This is a supporting file and is part of a larger project on wireless network optimization.
